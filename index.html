Portfolio Highlights

  • Chat Server

           C#       .NETframework       XML       WPF

    A program that sets up a server to host a chat application for clients. In the application users can create an account, play rock paper scissors, and private message. A seciton of pong is also available which demonstrates a prediction algorithm based off movement to replicate this across playing clients. A lobby system is used to match players against other players wanting to play the same game.

    Click here to see the GitHub Repo

  • Doomn't

           UE4       C++       Team Project

    A zombie shoot em up style game developed as a team of 6. For this project my role was to set up the player movement, the animations withing the game and the aim down sight system for the gun. The animation system was set up using a combination of Unreal Engine's animation blueprint and animation system and used animation notifies to help trigger events such as destroy the object if it died and also help with melee combat. The aim down sght system used unreal engines socket system to help move the gun to position the socket in front of the camera to create the effect of looking down the sight.

    Click here to see the GitHub Repo

  • Steering Bevahiours and Pathfinding

           C++       Univeristy Framework      

    Built on a framework provided by the University to showcase steering behavious and A* pathfinding using a car. The steering behaviours demonstrated were seek, arrive, flee, pursue, wander and object avoidance. To show case these the I changed the movement system in the frame work to use physics and forces to move the car which helped combine them by applying a total force accumulated through the use of each steering behaviour. The A* path finding used calculated using a link list to help the car find the quickest path to the passenger if the car had a certain amount of fuel remaining.
